당회록 양식 다운로드

매월 정기(또는 임시) 당회 회의록 작성에 사용하는 공식 양식입니다. 다운로드한 파일을 열어 일시·장소·참석자·논의 사항·결의 사항을 채운 뒤, 완성된 파일을 구글 드라이브의 당회록 폴더에 올려주세요. 검색 시스템이 주기적으로 새 파일을 자동으로 찾아 색인합니다.

⬇ 템플릿 다운로드 (.docx) 📁 구글 드라이브 폴더 열기
작성 순서
  1. 위 "템플릿 다운로드" 버튼으로 양식을 받습니다.
  2. 일시 / 장소 / 회의 종류 / 기도자 / 참석자를 입력합니다.
  3. 논의 사항과 결의 사항을 작성합니다.
  4. 서기 · 당회장 이름을 적습니다.
  5. "구글 드라이브 폴더 열기"로 이동하여, 해당 월/연도 폴더에 파일을 업로드합니다.
  6. 업로드한 당회록은 다음 자동 색인 시점부터 검색 탭에서 찾을 수 있습니다.

논의 사항 / 결의 사항 본문 내용을 기준으로 검색합니다. (Searches the content of 논의 사항 and 결의 사항 sections.)
